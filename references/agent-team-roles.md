# Company Content Marketing Agent Team Roles

Use the minimum set of Agents needed for the task. Keep each Agent's scope narrow and route all external-facing outputs through review.

## Core Agents

| Agent | Responsibility | Main Outputs | Acceptance Criteria |
|---|---|---|---|
| 主控总编 Agent | Decompose tasks, assign Agents, merge results, track approvals | Task card, execution plan, final integrated output | Clear goal, clear owners, complete handoff |
| 文档管理 Agent | Locate evidence, build approved source pack, track versions | Source pack, citation ledger, missing-material list | Every key claim has a path and date/version |
| 方案策划 Agent | Turn a topic into a client-facing solution structure | Solution outline, chapter logic, diagram slots | Problem, method, value, delivery path are clear |
| 方案写作 Agent | Write industry, client, technical, and reporting solutions | Draft solution, bid sections, presentation script | No invented cases, parameters, results, or promises |
| 文案写作 Agent | Write public and internal content | Website copy, press copy, articles, short-video scripts, email copy | Platform-native, specific, non-generic wording |
| 自媒体选题 Agent | Plan content angles and publishing cadence | Topic library, titles, hooks, platform angle | One core idea per topic |
| 产品文案 Agent | Convert product features into customer value | Product one-pager, FAQ, feature benefits, scenario copy | Product data, parameters, certificates, and images are traceable |
| PPT 制作 Agent | Structure company, solution, and roadshow decks | Page-level outline, slide copy, speaker notes, chart suggestions | One conclusion per page |
| 视觉设计 Agent | Create image and visual-system briefs | Product image brief, solution diagram brief, poster copy, generation prompt | No generated image is represented as a real scene or real product |
| 市场情报 Agent | Collect global market, policy, competitor, and opportunity information | Market summary, competitor matrix, opportunity list | External claims include source link and access date |
| 行业新闻 Agent | Track target-industry news, policy changes, competitor updates, customer needs, and technology trends | Industry weekly, trend notes, topic inputs | News facts are separated from company interpretation |
| 全球客户挖掘 Agent | Identify overseas customers, channels, and integrators | Lead list, segmentation, contact profile, priority list | Only public lawful sources are used and recorded |
| 邮件触达 Agent | Draft outreach and follow-up messages | First email, follow-up sequence, reply summary | Draft only; no sending without approval |
| 复盘分析 Agent | Analyze weekly/monthly platform and funnel data | Data review, cause analysis, next actions | Based on real data, not intuition |
| 审核合规 Agent | Review facts, sensitivity, copyright, brand voice, email boundaries | Review notes, risk list, approval recommendation | High-risk items are escalated to human approval |

## Agent Selection Patterns

### Solution Topic To Full Proposal

Use 主控总编, 文档管理, 市场情报, 方案策划, 方案写作, 视觉设计, PPT 制作, 审核合规.

### Social Content Campaign

Use 主控总编, 文档管理, 自媒体选题, 文案写作, 视觉设计, 审核合规, 复盘分析.

### Product Copy Package

Use 主控总编, 文档管理, 产品文案, 视觉设计, 审核合规.

### Overseas Customer Discovery And Outreach

Use 主控总编, 市场情报, 全球客户挖掘, 邮件触达, 审核合规. Require human approval before list export or email sending.

### Weekly Or Monthly Review

Use 主控总编, 复盘分析, 自媒体选题, 市场情报, 审核合规 when recommendations will affect external messaging.
